## 環境
> [!WARNING]
> 3.10.0 為 `開發環境` 其他版本未經測試
- `Python 3.10.0 (AMD64 on win32)` 


## 開始
### 安裝 [PyPI](https://pypi.org/project/cdps)
```
pip install cdps
```
### 初始化
```
cd ./my_cdps
cdps init
```
### 執行
```
cdps
```


## 更新
```
pip install --upgrade cdps
```
